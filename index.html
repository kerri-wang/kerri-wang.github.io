Global Advertising Performance Analysis

Analyzed 1,800 ad campaigns across Google Ads, Meta Ads, and TikTok Ads spanning 5 industries and 7 countries throughout 2024. The project covers end-to-end exploratory analysis, statistical hypothesis testing (ANOVA, Kruskal-Wallis, Mann-Whitney U with Cohen's d effect sizes), and strategic budget reallocation recommendations. Key findings include platform-specific ROAS patterns, industry and geographic efficiency gaps, and seasonal trends that inform campaign timing.

Tools: Python, Pandas, NumPy, Matplotlib, Seaborn, SciPy, Jupyter Notebook, GitHub

GitHub Repository
